# ⚡ NEXUS — 超越想象

一个极致酷炫的品牌展示网站，融合 3D 粒子系统、动态渐变、视差滚动与流体动画。

## ✨ 特性

- 🎯 **3D 粒子宇宙** — Three.js 实时渲染 4500+ 动态粒子
- 🖱️ **鼠标视差** — 多层次深度感知交互
- 🎨 **动态排版** — 渐变文字、描边特效、发光字体
- 🌊 **丝滑转场** — GSAP + ScrollTrigger 驱动
- 📱 **响应式设计** — 完美适配移动端

## 🚀 部署

```bash
# 创建 GitHub 仓库并推送
gh repo create nexus-site --public --source=. --remote=origin --push
# 在仓库 Settings > Pages 中启用 GitHub Pages
```
