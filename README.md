# ✦ 宋词宇宙 — 星辰中的千年绝唱

一个沉浸式 360° 3D 宇宙交互网站。双击进入浩瀚星空，每颗星辰都是一首千古宋词。

## ✨ 特性

- 🌌 **360° 宇宙漫游** — Three.js + OrbitControls，鼠标拖动旋转视角，滚轮缩放
- ⭐ **30 颗诗星** — 每颗星辰对应一首经典宋词，带有脉冲呼吸光效和中文标题标签
- 📜 **宋词品读** — 点击星辰弹出精美卡片，展示词牌名、作者、朝代、全文及简介
- 🎨 **极致视觉** — 8000 颗背景星云粒子 + 2000 颗内层辉光粒子 + 中央光晕
- 🖱️ **鼠标悬停放大** — 指向星辰时放大 1.8 倍，带弹性反馈
- 🔄 **自动旋转** — 默认自动漫游，交互时暂停
- ⚡ **纯前端** — ES Module + Import Map，零构建工具

## 🎯 操作

| 操作 | 效果 |
|------|------|
| 双击页面 | 进入宇宙 |
| 拖动鼠标 | 旋转视角 |
| 滚轮 | 缩放远近 |
| 点击星辰 | 品读宋词 |
| ESC / 点击背景 | 关闭词卡 |
| ← 返回 | 回到入口 |
